# 光学モデル仕様（optics-spec）

実装の物理的根拠をまとめる。Claude Code 等で各フェーズを実装する際の一次資料。
**単位は内部的にすべて SI（m / rad / D）に統一**し、UI 層でのみ mm・度・小数視力へ変換する。

---

## 1. 符号則

- 光は +z 方向（角膜頂点を z=0、後方が正）に進む。
- 曲率半径 R: 曲率中心が面より**後方（+z 側）にあれば正、前方なら負**。
  - 角膜前面・水晶体前面 → R 正
  - 水晶体後面（後方に凸）→ R 負
- 屈折: **n'·u' = n·u − y·Φ**, 面屈折力 **Φ = (n' − n) / R** [D]
- 転送: **y_next = y + u'·d**（d は物理距離 [m]、u' は屈折後の近軸スロープ）
- 眼屈折（角膜面基準の遠点バージェンス）: **正 = 遠視、負 = 近視**

---

## 2. 模型眼（有水晶体）

水晶体を 1 枚の薄レンズにせず**前面・後面の 2 面**で表す。これは眼内ガス／オイルの効果が
水晶体後面の屈折率差変化として現れるため、薄レンズ近似では再現できないことによる。

| 面 | 既定半径 [mm] | 後方媒質 | 屈折率 |
|---|---|---|---|
| 角膜 | 7.8 | 房水 | 1.336 |
| 水晶体前面 | 10.0 | 水晶体 | 1.42 |
| 水晶体後面 | −6.0 | 硝子体腔 | nVit（可変） |

距離: 前房深度 ACD = 3.6mm、水晶体厚 = 3.6mm、眼軸長 = 24.0mm（角膜頂点→網膜）。
媒質屈折率: 空気 1.0 / 房水・硝子体 1.336 / 水晶体 1.42 / ガス ≈1.0 / シリコンオイル ≈1.40。

> ⚠️ 既定半径は近似生理値で、正常眼が厳密に正視にならない（検証値で −0.74D）。
> 「校正」節に従い必要なら微調整する。本教材の目的は**絶対値の再現ではなく変化の向きと桁の理解**。

---

## 3. 眼屈折の解法

近軸系では網膜面での光線高さ Y は物体バージェンス Lobj の一次関数。よって 2 点（Lobj=0, 1）を
評価し **Y(Lobj)=0** となる Lobj を線形に解けば眼屈折が厳密に求まる（`ocularRefraction`）。

---

## 4. 検証済みの数値（実装テストの根拠）

| 条件 | 計算値 | 臨床的妥当性 |
|---|---|---|
| 正常眼（24mm） | 眼屈折 −0.74 D | ほぼ正視（要校正） |
| 後側焦点（正常眼） | 角膜頂点から ≈23.8mm | 網膜近傍 ✓ |
| 眼軸 24→25mm | **−2.62 D/mm** | 臨床通説 ≈ −2.7 D/mm と一致 ✓ |
| シリコンオイル充填 | **+7.91 D 遠視化** | Gullstrand 模型眼 ≈ +8D と一致 ✓ |
| ガス充填（有水晶体） | 強い近視化（向きが要点） | 完全充填中は視機能ほぼ消失 |

## 5. 乱視（スツルムの円錐）

2 主経線で角膜曲率半径を変えた 2 つの眼を作り、各後側焦点（前焦線・後焦線）を求める。
最小錯乱円は**ジオプター（バージェンス）空間の中点** 1/z_clc = (1/z_前 + 1/z_後)/2 の位置。
等価球面屈折 = (強主経線屈折 + 弱主経線屈折)/2。

---

## 6. タンポナーデ媒質の屈折シフト（重要・教育的核心）

水晶体後面屈折力 Φ_post = (nVit − n_lens) / R_post（R_post < 0）。

- **オイル（nVit=1.40 > 1.336）**: Φ_post 低下 → 系の屈折力低下 → 後側焦点が後退 → **遠視化**
- **ガス（nVit=1.0 < 1.336）**: Φ_post 増加 → 系の屈折力増加 → 後側焦点が前進 → **近視化**（オイルと逆向き）
- **無水晶体（phakic=false）＋オイル**: 水晶体面を除去するとオイル前面が凸メニスカスとなり
  **近視化に逆転**。シフト量は瞳孔径に依存（文献既知）。`makeEye({ phakic:false })` で扱う。

> 文献根拠: シリコンオイルは硝子体より屈折率が高く、模型眼で約 +8D の遠視化を生じる。
> 無水晶体眼ではオイルが凸面を形成し近視化に逆転する（瞳孔径依存）。
> 臨床の PPV 後の小さな近視化（≈ −0.5〜−1.6D）は主に IOL/水晶体–虹彩隔膜の位置変化によるもので、
> 媒質屈折率の光学効果とは区別して教える。

## 7. PSF と見え方（Phase 5）

- 瞳孔関数 **P(x,y) = A(x,y)·exp(i·2π/λ·W(x,y))**、A は開口、W は波面収差 [m]。
- **PSF = |FT{P}|²**。網膜像 = 物体 ⊛ PSF。
- 波面 W は Zernike 係数で与える（OSA/ANSI 順）。デフォーカス D[D] と瞳半径 r[m] の対応は
  W_defocus ≈ (D·r²)/2。OSA 正規化への厳密係数換算は Phase 5 で検証・確定する。
- ランドルト環: 小数視力 1.0 でギャップ 1 分角・環全体 5 分角。
  任意視距離は視角 = 2·atan(物理サイズ/2 / 距離) で換算。
- 検証: デフォーカス 0 で Airy パターン（回折限界）に収束、デフォーカス増で切れ目判別不能。

## 8. 校正手順

1. `npm test` で向きと桁を確認（必須）。
2. 正常眼を正視に近づけたい場合、角膜半径または眼軸長を微調整し、`backFocalDistanceFromCornea`
   が眼軸長に一致するよう合わせる。
3. オイル遠視化量を臨床値（+2〜+6D）に寄せたい場合は水晶体後面半径・水晶体屈折率を調整。
   ただし**変化の向きと桁の保持**を優先し、過度な作り込みは避ける。
